Напишите программу, которая рисует изображение правильных многоугольников по образцу. Многоугольники должны иметь разный цвет.

Примечание 1. Доработайте программу, чтобы площадь всех многоугольников была одинаковой. Площадь правильного многоугольника вычисляется по формуле \[S = \dfrac{na^2}{4\tg \left(\dfrac{180^{\circ}}{n}\right)}, \, \, где \, \, \, n - количество \, сторон, \, a - длина \, стороны.\]

Примечание 2. Обратите внимание на то, что тригонометрические функции модуля math работают с радианами, а не градусами. Для преобразования градусов в радианы используйте функцию radians.
