Вещественные числа

Как уже было сказано в уроке, объединение рациональных и иррациональных чисел образует множество вещественных чисел. Геометрическая интерпретация множества вещественных чисел – это числовая прямая:

Каждому вещественному числу соответствует определённая точка числовой прямой, и наоборот – каждой точке числовой прямой обязательно соответствует некоторое вещественное число. Другими словами, числовая прямая полностью покрыта вещественными числами.

Мнимые числа

Рассмотрим квадрат произвольного вещественного числа:

  • \(2 \cdot 2 = 4\)
  • \((-2) \cdot (-2) = 4\)
  • \(0 \cdot 0 = 0\)
  • \(0.1 \cdot 0.1 = 0.01\)

Результат всегда неотрицательное число. Действительно, пусть $a$ – произвольное вещественное число. Тогда $a \cdot a = a^2 \ge 0$. Таким образом, на множестве вещественных чисел, перемножая два одинаковых числа мы всегда получаем неотрицательное число (большее, либо равное нулю).

Представим, что существует число $i$ (не являющееся вещественным), такое, что $i\cdot i = i^2 = -1$. Будет ли такое обозначение полезным? Оказывается, да!

Такие числа в математике называются мнимыми и обозначают их буквой $i$, от слова imaginary (мнимый, воображаемый). Итак, получаем:\[i^2 = -1 \Longrightarrow i = \sqrt{-1}.\]

С таким числом (обозначением), можно решать задачи, где требуется извлечь квадратный корень из отрицательного числа.

Рассмотрим несколько примеров.

Пример 1. Выразите число $\sqrt{-9}$ через $i$.

Решение. Согласно введенному обозначению можем записать:\[\sqrt{-9} = \sqrt{9}\cdot \sqrt{-1} = 3i\]

Или в общем виде для любого положительного числа $x$ существует равенство:\[\sqrt{-x} = \sqrt{x}\cdot i\]

Пример 2. Посчитать значение выражения $(5i)^2$.

Решение. Согласно определению мнимой единицы $i^2 = -1$, получаем:\[(5i)^2= 25i^2 = 25\cdot (-1) = -25.\]

Пример 3. Решите уравнение $x^2 + 1 = 0$.

Решение. Запишем уравнение в виде $x^2 =-1$. Такое уравнение не имеет решений на множестве вещественных чисел. Однако уравнение имеет решения на множестве мнимых чисел. Имеем:\[x^2 = -1 \iff x = \pm \sqrt{-1} = \pm i.\]

Мнимая единица

Число $i = \sqrt{-1}$ называется мнимой единицей. Введение мнимой единицы позволяет расширить множество вещественных чисел до множества комплексных чисел.

Примеры мнимых чисел:

$i$ $11.67i$ $-i$ $3i/4$ $0.02i$ $\pi i$

Мнимая единица $i$ обладает интересным свойством: каждый раз при умножении на $i$ она "циклически" проходит через 4 различных значения:\[1 \cdot i = i\\ i \cdot i = -1\\ -1 \cdot i = -i \\ -i \cdot i = 1\]

Получаем равенства натуральных степеней мнимой единицы:\[i = \sqrt{-1}, \, i^2 = -1, \, i^3 = i^2 \cdot i = -i, \, i^4 = i^3 \cdot i = 1, \, i^5 = i^4 \cdot i = i, \ldots\]

Несложно посчитать значения для больших показателей:\[i^{100}= (i^2)^{50} = (-1)^{50} = 1, \, i^{55} = (i^2)^{27} \cdot i = -i.\]

Комплексные числа в математике

Комплексное число – комбинация вещественного и мнимого числа. Таким образом, число вида $a+bi$, где $a, b$ – вещественные числа, называется комплексным числом.

Число $a$ – называется вещественной (действительной) частью комплексного числа, а число $b$ – мнимой частью числа.

Примеры комплексных чисел:

$1+i$ $39+4i$ $0.5-6.7i$ $-2+\pi i$ $\sqrt{2} + \sqrt{3} i$

Заметим, что действительная (число $a$) и мнимая (число $b$) часть комплексного числа $a + b i$ может быть равна нулю:

Комплексное число Действительная часть Мнимая часть Комментарий
$3+2i$ $3$ $2$  
$7$ $7$ $0$ вещественное число
$-5i$ $0$ $-5$ мнимое число

Можем ли мы конструировать одно число на основании двух других? Конечно, можем! И уже делали это, когда вводили рациональные числа. Дробь $\dfrac37$ – конструкция на основании двух натуральных чисел $3$ и $7$. Комплексное число – конструкция на основании вещественного и мнимого числа.

Комплексным числам нет места на числовой прямой, для них нужна так называемая комплексная плоскость. Для графической интерпретации комплексных чисел нам потребуется две взаимно перпендикулярные прямые.

Горизонтальная прямая (ось) отвечает за действительную часть комплексного числа, вертикальная – за мнимую.

Обозначения комплексных чисел

Комплексные числа обычно обозначаются буквой $z = a+bi$. Действительная часть комплексного числа обозначается как $Re(z) = a$, мнимая часть обозначается $Im(z) = b$.

Модуль комплексного числа

Модулем комплексного числа $z = a +bi$ называется вещественное число, равное \[|z| = \sqrt{a^2+b^2}\]Такое число, равно расстоянию от точки $(a; \, b)$ до начала координат, то есть до точки $(0; \, 0)$.

Сложение комплексных чисел

Пусть $z_1 = a+ bi, \, z_2 = c + di$. Тогда \[z_1+z_2 = (a+bi)+ (c+di) = (a+c)+(b+d)i\]Пример. Пусть $z_1 = 3 + 2i, \, z_2 = 1 + 7i$, тогда 

\[z_1 + z_2 = (3 + 2i) + (1 + 7i) = (3 + 1) + (2 + 7)i = 4 + 9i.\]

Разность комплексных чисел

Пусть $z_1 = a+ bi, \, z_2 = c + di$. Тогда \[z_1- z_2 = (a+bi) - (c+di) = (a-c)+(b-d)i\]Пример. Пусть $z_1 = 3 + 2i, \, z_2 = 1 + 7i$, тогда 

\[z_1 - z_2 = (3 + 2i) - (1 + 7i) = (3 - 1) + (2 - 7)i = 2 - 5i.\]

Умножение комплексных чисел

Пусть $z_1 = a+ bi, \, z_2 = c + di$. Тогда \[z_1\cdot z_2 = (a+bi) \cdot (c+di) = ac+adi+bci+bdi^2 = (ac-bd)+(ad+bc)i.\]   При умножении комплексных чисел возникает $i^2$, значение которого равно $-1$.

Пример 1. Пусть $z_1 = 3 + 2i, \, z_2 = 1 + 7i$, тогда 

\[z_1 \cdot z_2 = (3 + 2i) \cdot (1 + 7i) = (3- 14)+(21+2)i = -11+23i.\]

Пример 2. Пусть $z = 1+i$, тогда

\[z ^2 = z\cdot z = (1+i)\cdot (1+i) = (1-1) + (1+1)i = 0 +2i = 2i.\]

Сопряженные комплексные числа

Число $\bar{z} = a - bi$ называется сопряженным к числу $ z = a+bi$. Произведение двух сопряженных чисел является вещественным числом. Действительно:\[z \cdot \bar{z} = (a+bi)\cdot (a-bi) = (a^2+b^2)+(-ab+ab)i = a^2+b^2.\]

    При умножении сопряженных комплексных чисел возникает $i^2$, значение которого равно $-1$.

Деление комплексных чисел

Пусть $z_1 = a+ bi, \, z_2 = c + di$. Тогда \[\dfrac{z_1}{z_2} = \dfrac{a+bi}{c+di} = \dfrac{(a+bi)(c-di)}{(c+di)(c-di)} = \dfrac{(ac+bd)+(bc-ad)i}{c^2+d^2} = \dfrac{ac+bd}{c^2+d^2} + \dfrac{bc-ad}{c^2+d^2}i.\]Таким образом, для того, чтобы поделить два комплексных числа, мы умножаем числитель и знаменатель дроби на число сопряженное к знаменателю, в результате в знаменателе оказывается вещественное число!

    При делении комплексных чисел возникает $i^2$, значение которого равно $-1$.

Пример. Пусть $z_1 = 3 + 2i, \, z_2 = 1 + 7i$, тогда 

\[\dfrac{z_1}{z_2} = \dfrac{3+2i}{1+7i} = \dfrac{(3+2i)(1-7i)}{(1+7i)(1-7i)} = \dfrac{(3+14)+(-21+2)i}{1^2+7^2} = \dfrac{17-19i}{50} = \dfrac{17}{50}-\dfrac{19}{50}i.\]
